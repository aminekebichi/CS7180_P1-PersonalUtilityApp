# Let's Cook

**Members:** Amine Kebichi

## Project Description

**Let's Cook** is a tool for amateur chefs who would like to learn new recipes. It takes an input of what ingredients/groceries you currently have in stock and uses this information to suggest a recipe for you to try.

### Key Features
- **Recipe Suggestions:** Suggests recipes based on available ingredients.
- **Recipe Log:** Stores a log of all recipes you have made.
- **Smart Suggestions:**
    - Suggests new recipes to try as you continue to use the application.
    - Suggests recipes even if a few ingredients are missing options.
- **Chef Level:** Maintains a "chef-level" to suggest easier recipes for new users and incrementally suggest more complex/difficult recipes as the user logs more entries.
- **Filtering:** Options to filter suggestions by cuisine (Italian, Indian, Latin, etc.) and dietary specifications (Gluten-free, Pescatarian, Vegetarian, etc.).

## User Personas
- **The Novice Cook:** People who want to learn how to cook but don't know where to start.
- **The Cookbook Alternative:** Users looking for a more dynamic and personalized way to find recipes than traditional cookbooks.

## User Stories
1. **The Solo Cook:** "As someone who is recently living alone, I want some help coming up with ideas for what to cook."
2. **The Aspiring Chef:** "As an aspiring chef, I would like to learn new recipes and build a strong foundation in the culinary field."
3. **The Budget Student:** "As a student, I want to find easy recipes to make with limited resources available."
